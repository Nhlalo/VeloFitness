# 💪 Velo Fitness - Modern Fitness Website

A modern, responsive gym website built as a portfolio project showcasing fitness-focused web design, membership management, and payment integration.

![Gym Website Preview](/images/preview.png) <!-- Add a screenshot later -->

## ✨ Features

- **Responsive Design** - Mobile-first approach, works on all devices
- **Member Dashboard** - Account management, billing history, profile settings
- **Payment Integration** - Stripe test mode for membership signups
- **Location Finder** - ZIP/postal code search for gym locations
- **Class Schedule** - Interactive class booking interface
- **Authentication Flow** - Login/signup with validation

## 🛠️ Technologies Used

- React.js
- Stripe API (Test Mode)
- CSS3 / Tailwind CSS
- React Router
- TypeScript
